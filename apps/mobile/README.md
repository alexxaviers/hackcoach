# Simon Mobile (Expo)

Run locally:

1. Install deps: `cd apps/mobile && npm install` (or yarn/pnpm)
2. Start Metro: `npm start`
3. Run on device/simulator via Expo dev tools.

Configure `API_URL` and RevenueCat keys in `.env`.

Build with EAS: configure `eas.json` and run `eas build` for iOS (TestFlight) or Android (Internal Testing).
